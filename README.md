# Adding Basketball, to Minecraft!

Simple Basketball adds Basketball into Minecraft! Get together with some friends and start hooping. This mod is currently "simple" as the name suggests, but does add some neat features (more planned in future!) (Now available in Fabric as well as Forge and NeoForge).

## Features:

*   Basketball
*   Throwing
*   Catching
*   Hoop

## Compatibility/Requirements:

Simple Basketball should be compatible with most mods, and has no requirements/library mods, although **[ItemPhysic](https://modrinth.com/mod/itemphysic) is strongly recommended** for more accurate ball physics. However, we do recommend a few mods to make your Basketball experience better:

*   [ItemPhysic](https://modrinth.com/mod/itemphysic)
*   [ParCool!](https://modrinth.com/mod/parcool)
*   [Real Camera](https://modrinth.com/mod/real-camera)
*   [Camera Overhaul Reforged](https://modrinth.com/mod/camera-overhaul-reforged)
*   And any performance mods you would like!
